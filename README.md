# 叨叨
哔哔点啥的白嫖版本。基于github的issue及vercel api搭建。
___

### config设置：
```yml
setting:
  zone: 8 #时区校准
  search_time_limit: 3 #默认最大查询时间范围3天
  search_time_limit_num: 5 #默认查询条数5条
  user: 'zfour' #用户名
  repo: 'daodao'  #用户仓库
```

### vercel环境变量设置:
